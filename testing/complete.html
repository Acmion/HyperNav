HyperNav Complete

This an example that showcases nearly everything that HyperNav is capable of. The layout is relatively complex and contains some slight inconsistencies (such as different types of dropdown menus) for complete coverage. This layout should be extended if needed and used as the main testing HyperNav.

This example is not completely keyboard accessible, due to the usage of modes in the top navigation bar. See more information in the documentation.

Note: No JavaScript, except a CSS variable ponyfill for IE11, is utilized on this page. Everything is built with HTML and CSS.

Details

This section describes how the behavior of various items differ from each other.

Top Navigation Menu

Home

Navigation item with an icon. Is always visible, regardless of device.

About

Navigation item without an icon.

Products

Dropdown menu that expands to the right. Different expand icon for mobile and desktop. Split button.

Team

Dropdown menu that expands to the center. Full button.

Shipping

Dropdown menu that expands to the left. Delay animation. Split button.

Settings (Cog)

A mode that reveals new items below the top navigation bar.

Search (Magnifying Glass)

A mode that overrides the top navigation bar.

Menu (Three Bars)

Hides and reveals the main top navigation content on mobile devices. Only visible on mobile.

Side Navigation Menu

Collapse

Collapse and expand the side navigation menu.

Space Exploration

Navigation item with an icon.

Nebulas

Dropdown menu with one sublevel.

Stars

Dropdown menu with several sublevels.
