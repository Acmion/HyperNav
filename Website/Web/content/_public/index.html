HyperNav



HyperNav is an advanced, interactive and open source navigation menu framework without JavaScript. Build and display complex site navigation structures for both mobile and desktop users.

Download Documentation GitHub


Responsive

Designed for both desktop and mobile devices.

Modular

Built with SCSS, so just import what you need.

Prefixed CSS

The CSS classes are prefixed, thus HyperNav works well together with other CSS frameworks.

CSS Only

HyperNav does not use any JavaScript or other scripting languages and is solely built on CSS.

Lightweight

The payload of HyperNav is less than 4KB when gzipped.

Basic Keyboard Accessibility

HyperNav supports basic keyboard accessibility without JS.
